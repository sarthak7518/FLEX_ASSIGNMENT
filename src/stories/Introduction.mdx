# FlexPrice Component Library

A small, opinionated React component kit themed for **FlexPrice** — a usage-based
billing & pricing surface. Designed as a frontend take-home for the intern role.

This Storybook ships **16 components**, organized by purpose:

### Foundations
- `Button` · `Badge` · `Avatar` · `Input` · `Toggle` · `Spinner` · `ProgressBar` · `Alert` · `Tooltip`

### Pricing
- `PriceTag` · `PlanCard` · `FeatureList` · `MetricCard`

### Layout
- `Card` · `Tabs` · `Modal` · `Pagination`

### Design language
- **Display font**: Fraunces (variable serif) — used for prices and titles
- **UI font**: Inter Tight
- **Mono**: DM Mono — used for labels and eyebrows
- Warm off-white `#f7f7f5` canvas with **ink** (`#0b0c0f`), **accent orange** (`#ff5b1f`),
  **mint** (`#c9f27a`) and **violet** (`#5b3df5`) accents.

### Conventions
- Each component is a single-responsibility primitive with plain CSS variables.
- Variants are exposed via `variant`, `tone`, `size` props.
- All stories include `argTypes` controls so reviewers can poke at edge cases.

> Use the sidebar to navigate. Each component has multiple stories covering the
> common variants and an interactive **Playground** with Storybook Controls.
